get_data(bounding_box: boundingbox) → Tuple[GeoDataFrame, GeoDataFrame][source]#

Downloads and processes OSM data for the given bounding box.

Parameters:

bounding_box (boundingbox) – The bounding box for which to download data.

Returns:

A tuple containing two GeoDataFrames: - nodes (geopandas.GeoDataFrame): GeoDataFrame containing point features (e.g., nodes). - edges (geopandas.GeoDataFrame): GeoDataFrame containing line and polygon features (e.g., edges).

Return type:

tuple

Raises:

ValueError – If an error occurs during the OSM data download or processing.

Notes

The method uses the osmnx library to fetch OSM data and filters it into nodes and edges based on geometry types.
