get_green(**kwargs)[source]#

Filters and processes green areas from the Copernicus data.

Args:
**kwargs: Additional arguments to specify green areas.

  • green_areas (frozenset, optional): A set of values representing green areas (default is frozenset([10, 20, 30])).

Returns:
dict: A dictionary containing the filtered green areas with the following keys:

  • ‘data’: The binary raster data where green areas are marked as 1.

  • ‘transform’: The affine transform of the raster.

  • ‘crs’: The coordinate reference system of the raster.

  • ‘shape’: The shape of the raster.

get_green(**kwargs)[source]#

Filters and processes green areas from the OSM data.

Args:
**kwargs: Additional arguments to specify green tags.

  • green_tags (dict, optional): A dictionary of tags to filter green areas. Default is: {

    ‘natural’: [‘wood’, ‘tree_row’, ‘tree’, ‘scrub’, ‘grassland’,

    ‘heath’, ‘fell’, ‘tundra’, ‘shrubbery’],

    ‘landuse’: [‘forest’, ‘meadow’, ‘grass’, ‘allotments’], ‘leisure’: [‘park’, ‘garden’, ‘nature_reserve’]

    }

Returns:
tuple: A tuple containing two GeoDataFrames:

  • green_nodes (geopandas.GeoDataFrame): Filtered green nodes.

  • green_edges (geopandas.GeoDataFrame): Filtered green edges.

Raises:

Exception: If an error occurs during processing, returns empty GeoDataFrames.
